Task 1: Personal Portfolio Website
Create a personal portfolio showcasing your skills, projects, and resume using HTML, CSS, and JavaScript. Your portfolio should have a clean and professional design, making it easy for visitors to navigate and understand your expertise. Use HTML for structuring the content, CSS for styling to make it visually appealing, and JavaScript to add interactivity, such as smooth scrolling, dynamic content loading, or animations. Include sections such as About Me, Skills, Projects, Resume/CV, and Contact information. Make sure your portfolio is responsive and works well on various devices, including desktops, tablets, and smartphones.

Task 2: Expense Tracker
Develop an expense tracker using HTML, CSS, and JavaScript. This web application should allow users to add, edit, and delete expenses. Implement a user-friendly interface with forms for adding new expenses, buttons for editing or deleting existing ones, and a display area for listing all expenses. Utilize JavaScript to handle user interactions, such as validating input data, performing calculations, and updating the UI dynamically. Incorporate local storage to store user data persistently, ensuring that expenses are saved even when the user closes or refreshes the browser. Make the expense tracker intuitive and efficient for managing personal finances.

Task 3: Social Media Dashboard
Design and build a social media dashboard that aggregates content from various social media platforms. Use HTML, CSS, and JavaScript to create a visually appealing and interactive interface. Integrate APIs or widgets provided by social media platforms to fetch and display content such as posts, images, videos, and statistics. Organize the dashboard to present information from different platforms in a structured and intuitive manner, allowing users to easily navigate between feeds and interact with posts. Implement features such as filtering, sorting, and searching to enhance usability. Ensure that the dashboard is responsive and works smoothly across different devices and screen sizes, providing a seamless user experience.
